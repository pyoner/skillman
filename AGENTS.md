# Agentic Coding Guidelines - skillman

`skillman` is a CLI tool that converts CLI help text and manual pages into the **Agent Skill** specification format using **Bun** and **TypeScript**.

### Core Commands

- **Install Dependencies**: `bun install`
- **Run App**: `bun run src/index.ts`
- **Type Check**: `bun x tsc --noEmit`
- **Test**: `bun test`

### Directory Structure

- `src/`: Main source code.
- `src/index.ts`: CLI entry point.
- `src/lib/`: Reusable logic and utilities.
- `tests/`: Test files.
